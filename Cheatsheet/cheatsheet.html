Table Tags

Tags Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attributes Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

How to Use

First, to set up your table, you'll need to build your HTML skeleton for the table. Think of the table as a mini HTML document. Declare your table using the "table" tag. Next, your table will need a head and a body, just like a full HTML document. This can be accomplished using the "thead" and "tbody" tags. Within the "thead" section, you'll need to name the coloumns of the table. (Note: Table columns span left to right while table rows span top to bottom)

CSS Selectors

Syntax Basic Selectors Description
* {property:value;} Universal Selects all elements on the pages.
p {property:value;} Type Selects all HTML tag/elements of a given type in your document.
#id {property:value} ID Selects an element based on the value of its unique ID attribute.
.class {property:value;} Class Selects all elements in the document that have the given class attribute.
a[attribute=value] {property:value;} Attribute Select all elements that have a specified attribute.
selector1 selector2/ selector1 + selector2 / selector1 > selector2 {property:value;} Combinators Complex selectors consisting of more than one selectors having some relationship between them.
selector: pseudo-class {property:value;} Pseudo Define the special state of an element to add an effect to an existing element based on its states.
